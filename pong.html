Pong: The Musical

This project was a simple one for myself. The goal here is to create a simple game that runs on nice logic and is an easy target for myself. The reason being is that most projects end soon after the idea because sometimes they are over complicated. I wanted to create a number of games small in scope so I can show off (my many talents). This project was to create a nice feeling and simple game but with one huge difference. The games sound was reactive. There are some games (mostly those spoopy ones ) that have a reactive soundtrack often used to invoke fear or surprise. This however creates a unique soundtrack specific to how both players (Or player and cpu) are playing the game. Hopefully it peaks an interest for the music and game nerds out there.

Finally we gotta reset it somehow and then we got our game.
